Яка команда дозводить швидко створити новий React проект?

Для швидкого створення нового проекту на React використовується команда npx create-react-app myReactApp, де myReactApp - назва проекту.

Щоб скориситатись цією командою нам потрібно встановити Node версії 8.10+ та npm версії 5.6+, npx - інструмент для запуску пакетів в npm.

Для того, щоб запустити застосунок в режимі розробки треба перейти в директорію проекту, виконавши команду cd myReactApp, та виконати команду npm start.
